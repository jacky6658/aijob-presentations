隱私政策

最後更新：2026年2月12日

1. 資料收集

本應用程式（Threads發文）僅收集您授權的 Threads 帳號資訊，用於發佈內容至您的 Threads 帳號。

2. 資料使用

我們收集的資料僅用於：

  • 透過 Threads API 發佈您授權的內容
  • 維護應用程式正常運作

3. 資料儲存

我們不會儲存您的個人資料或 Threads 內容。所有操作均透過 Meta Threads API 即時處理。

4. 第三方分享

我們不會將您的資料分享給任何第三方。

5. 您的權利

您可以隨時撤銷本應用程式的授權，方法為：

  • 前往 Instagram/Threads 設定
  • 選擇「帳號中心」→「你的資訊和權限」→「已連結的體驗」
  • 移除「Threads發文」應用程式

6. 聯絡我們

如有任何隱私相關問題，請聯絡：jacky051285@yahoo.com.tw

7. 政策變更

我們可能會不定期更新本隱私政策。重大變更將透過應用程式通知您。
