⚠️ 預期挑戰與解決方案

挑戰 1：柬埔寨治安疑慮

候選人疑慮：柬埔寨詐騙事件影響，擔心安全
解決方案：
✓ 強調公司合法性（投審會核准、50年歷史、上市櫃客戶）
✓ 工廠位於金邊市區（非偏遠地區）
✓ 台灣團隊支援（不是孤軍奮戰）
✓ 提供安全區域居住建議

挑戰 2：家庭因素阻力

候選人疑慮：配偶/家人不願意外派柬埔寨
解決方案：
✓ 強調外派福利（返台機票、住宿補貼）
✓ 協助家庭溝通（提供柬埔寨生活資訊）
✓ 彈性工作安排（可能的短期外派選項）

挑戰 3：產業轉換疑慮

候選人疑慮：紡織業 → 時尚配件業，擔心不適應
解決方案：
✓ 強調製造業本質相同（成本結構類似）
✓ 財會技能通用（合併報表、成本核算、資金管控）
✓ 客戶更國際化（歐美零售大廠，學習機會多）
