## 四、YQ2 HR 接案管理詳細規劃

### 4.1 需要串接的 API

```
後端 URL：https://aijobcasebackend.zeabur.app

API 端點：
├── POST /api/ai/import     → 新增案件
├── GET  /api/ai/leads      → 查詢案件
├── POST /api/ai/update     → 更新案件
├── POST /api/ai/delete     → 刪除案件
├── POST /api/ai/progress   → 更新進度
├── POST /api/ai/cost       → 記錄成本
├── POST /api/ai/profit     → 記錄利潤
└── POST /api/ai/attachment → 上傳附件
```

### 4.2 定期任務（Cron Job）

| 時間 | 任務 | 說明 |
|------|------|------|
| 每天 09:00 | 案件進度檢查 | 有哪些案子該跟進？ |
| 每天 09:00 | Deadline 提醒 | 即將到期的交付 |
| 每週一 09:00 | 報價追蹤 | 還沒回覆的報價 |
| 每月 1 日 | 月度報表 | 上月收入/成本統計 |

### 4.3 通知機制

```
YQ2 HR 檢查到待辦事項
         ↓
透過 Telegram 通知 Jacky
         ↓
格式範例：
📋 今日待辦（2026-02-10）

【需跟進】
• 客戶 A - 報價已發 3 天未回覆
• 客戶 B - 專案進度 80%，明天交付

【Deadline】
• 客戶 C - 2/12 交付第一版

【財務】
• 本月已收：NT$ 50,000
• 待收款：NT$ 30,000
```
