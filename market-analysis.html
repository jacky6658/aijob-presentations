🔍 四、人選搜尋策略 (Sourcing Strategy)

🎯 AI工程師 (2人，80k-120k)

📋 目標人選畫像

  • 技能：Python + Machine Learning / 深度學習框架 (TensorFlow, PyTorch)
  • 經驗：3-5 年 AI/ML 專案經驗，有實際模型部署經驗佳
  • 背景：資工、資科、統計、數學相關科系
  • 加分項：有 Kaggle 競賽經驗、GitHub 開源專案、技術部落格

🔎 搜尋策略

  • GitHub：搜尋關鍵字「machine learning taiwan」「deep learning taipei」，語言篩選 Python，活躍度 >50 commits/年
  • LinkedIn：職稱「AI Engineer」「Machine Learning Engineer」，地區「Taipei」，3-5 年經驗
  • CakeResume：技能標籤「Python」「TensorFlow」「PyTorch」「深度學習」
  • 社群：AI/ML 台灣社群 (Facebook, Meetup)，研討會講者與積極參與者

📊 預估人才池

  • 台灣 AI 工程師總數約 3,000-5,000 人，符合 3+ 年經驗約 1,500 人
  • 扣除在職穩定者，潛在活躍求職者約 200-300 人
  • 建議目標：初步篩選 30-50 份履歷，深度洽談 10-15 人，推薦 5-8 人

💻 全端工程師 (3人，70k-110k)

📋 目標人選畫像

  • 技能：React + Node.js + TypeScript + 資料庫 (PostgreSQL/MySQL)
  • 經驗：2-4 年全端開發經驗，至少參與過 2 個完整專案
  • 背景：資工、資管相關科系，或自學轉職者（作品集優秀）
  • 加分項：有 AWS/GCP 部署經驗、CI/CD 經驗、開源貢獻

🔎 搜尋策略

  • GitHub：搜尋「react node taiwan」「fullstack developer taipei」，語言篩選 JavaScript/TypeScript
  • LinkedIn：職稱「Full Stack Developer」「Full Stack Engineer」，技能「React」「Node.js」
  • CakeResume：技能組合「React + Node.js + TypeScript」，經驗 2-4 年
  • 104/1111：反向搜尋「全端工程師 台北」，主動聯繫開放求職者

📊 預估人才池

  • 全端工程師市場供給充足，台北地區約 5,000-8,000 人
  • 符合 2+ 年經驗約 3,000 人，開放求職者約 500-800 人
  • 建議目標：初步篩選 50-80 份履歷，深度洽談 20-30 人，推薦 10-15 人

🚀 五、競爭分析與機會點

💪 我們的優勢

  • AI 自動化工具：YQ1 + YQ2 協作，履歷自動匹配、BD 自動化，效率提升 30-40%
  • 產業專注：深耕 AI、資訊軟體、半導體產業，建立專業形象
  • 數據驅動：Google Sheets + Dashboard 即時追蹤，決策有據
  • 快速回應：自動化流程縮短回應時間，提升候選人與客戶體驗

⚠️ 潛在挑戰

  • 品牌知名度低：Step1ne 為新創獵頭品牌，需建立信任
  • 人才競爭激烈：AI/全端工程師為熱門職位，大型獵頭公司資源豐富
  • 客戶數量少：目前僅 26 家潛在客戶，需快速擴展
  • 成交週期長：獵頭業務從接觸到成交平均 1-3 個月

🎯 市場機會點

🔥 機會 1：醫療AI興起

  • 康統醫學、醫知彼等醫療AI公司正在快速擴張
  • 傳統獵頭對醫療AI領域不熟悉，我們可以切入
  • 行動：深耕醫療AI產業，成為該領域的專業獵頭

🔥 機會 2：資安人才短缺

  • 杜浦數位、智影顧問等資安公司需求強烈
  • 台灣資安人才缺口大，供不應求
  • 行動：建立資安人才庫，與資安社群建立關係

🔥 機會 3：半導體產業回流

  • 矽格、怡凡得等半導體公司擴廠，人才需求增加
  • 政府政策支持，產業前景看好
  • 行動：拓展新竹科學園區客戶，專攻半導體人才
